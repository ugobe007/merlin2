## 💡 KEYNOTE-SPECIFIC TIPS

### Converting from PowerPoint:

- Open .pptx file in Keynote (File → Open)
- Some formatting may shift - review carefully
- Fonts may need adjustment
- Animations may need to be rebuilt

### Design System in Keynote:

- Create master slides with your colors
- Set up color palette: #1a1a1a (dark), #00ff88 (accent)
- Use SF Pro or Montserrat for headings
- Use Inter or Helvetica for body text

### Exporting:

- Export as PDF for leave-behinds
- Export as PowerPoint if investor uses Windows
- Export as Keynote for your own use
- Keep all three versions
