## Image Specifications
- **Format:** JPEG or PNG (JPEG preferred)
- **Dimensions:** 1200×800 px minimum, landscape or tall product shot
- **File Size:** Under 300 KB
- **Background:** White/clean preferred for product shots
- **Branding:** Vendor branding OK — use with vendor permission

## Getting Vendor Images
1. **ChargePoint:** Press kit at chargepoint.com/media
2. **Tesla:** tesla.com/presskit (Supercharger section)
3. **ABB:** e-mobility.abb.com/media
4. **Tritium:** tritiumcharging.com/media
5. **Siemens:** siemens.com/press
6. **Legrand:** legrand.us/resources
7. **Kempower:** kempower.com/media

**Always get written approval before using vendor images in customer-facing documents.**
