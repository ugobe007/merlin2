# USPTO Provisional Filing Checklist

## What the USPTO says a provisional filing needs

Per the USPTO's provisional application guidance, a provisional application needs:

- a written description complying with 35 U.S.C. 112(a)
- drawings if necessary to understand the invention
- the basic filing fee
- a provisional cover sheet identifying the application as provisional, all inventors, inventor residences, title, correspondence address, and any government interest

## Format

- For provisional filings, USPTO guidance states the written description and drawings can be filed electronically in Patent Center using PDF.
- The DOCX surcharge guidance applies to new utility nonprovisional applications under 35 U.S.C. 111(a), not to provisionals.

## Per-Application Filing Packet

For each Merlin group, prepare:

1. Cover sheet
2. Specification PDF
3. Drawings PDF
4. Payment

## Before You File

- Confirm the exact inventors for that group
- Confirm title consistency between the cover sheet and specification
- Remove internal-only notes from the spec
- Make sure all material you may later want to claim is disclosed
- Make sure any necessary drawings are included on filing day
- Confirm the correct small-entity or micro-entity status before paying fees

## Filing Day Checklist

- Open Patent Center
- Start a new provisional application
- Upload the cover sheet
- Upload the specification PDF
- Upload the drawings PDF
- Review metadata and document descriptions
- Pay the fee
- Download and save the filing receipt and submission package
