# Merlin Provisional Filing Docs

These documents are draft filing materials for four provisional patent applications intended for electronic filing through USPTO Patent Center.

## Assumption

These are drafted as **provisional applications under 35 U.S.C. 111(b)**. If you intend to file nonprovisional utility applications instead, the document package must change substantially because nonprovisional filings require claims and have different structured filing expectations.

- `group1_core_platform_provisional_spec.md`

- `group4_expansion_products_provisional_spec.md`

## Practical Filing Workflow

For each of the four groups:

1. Finalize inventor names and inventorship for that specific group.
2. Review the specification and expand any implementation details that are only implied.
3. Prepare drawings from the "Brief Description of the Drawings" section.
4. Convert the finalized specification to PDF.
5. File in Patent Center as a new provisional application with:
   - provisional cover sheet
   - specification PDF
   - drawings PDF
   - filing fee

## Important Notes

- A provisional is only as strong as its written support. Add any implementation detail you may want to claim later.
- Do not rely on the title alone. Make sure each specification contains the full architecture you may later want to claim.
- If a drawing is necessary to understand the invention, file it with the provisional.
- Do not file an IDS with a provisional.
