# Merlin Patent Filing Prep

This folder contains attorney-facing prep materials organized into four patent groups. The goal is to make each filing easier to draft by bundling related inventions into coherent families, carving out trade secrets, and listing the specific code and evidence that support reduction to practice.

## Files

- `00_master_filing_strategy.md` - portfolio-wide approach, timing, and handoff checklist
- `01_group_core_platform.md` - core quoting and wizard platform family
- `02_group_pricing_compliance.md` - pricing, compliance, audit, and quote integrity family
- `03_group_intelligence_pipeline.md` - market intelligence, RSS, ML, and opportunity-detection family
- `04_group_expansion_products.md` - advanced product modules, monetization, and platform expansion family

## Working Assumptions

- Group 1 represents the first/core filing family and is included here so the full portfolio stays internally consistent.
- These documents are filing-prep memos, not finished patent applications.
- Exact thresholds, weights, multipliers, vendor criteria, and calibration values should remain trade secrets unless counsel advises otherwise.
- The source material for these memos is primarily `IP_DISCLOSURE.md` plus referenced code paths in the Merlin repository.

## Recommended Use

1. Start with `00_master_filing_strategy.md`.
2. Give counsel the group memo for the next filing family.
3. Attach screenshots, code excerpts, architecture diagrams, and dated launch evidence to the selected memo.
4. Convert each memo into a provisional or nonprovisional draft with claims tailored to the cleanest architecture-level story.
